Voraussetzungen

Für die Fast-Fourier-Transformation kommt die Bibliothek FFTW in der Version 2.x zum Einsatz. Diese muss vorher in compilierter Form vorliegen.

Konfiguration

Das Makefile für OS/2 gcc Version 3.3 geschrieben und enthält zudem den Pfad zur rfftw-Library. Kurzum, da ist Feintuning erforderlich. Eine Anpassung der Pfade und Compiler-Optionen sollte aber im allgemeinen hinreichend sein.

Kompilieren

cd src
make

Wenn alles passt, werden beim Aufruf von make im Verzeichnis bin alle ausführbaren Dateien erstellt.
